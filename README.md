# ITChatbot
Chatbot For Solving IT Problem On Top Of Django

Please install all Python package that written below in a virtual environment.
1. Django
2. nltk
3. keras
4. pickle

Enter the virtual environment and run the command below:

python manage.py runserver

Check whether there is any error.
If everything is fine, there will be output showing IP that you will need to copy and paste to browser to start the chatbot program.
Google chrome and Edge recommended.
